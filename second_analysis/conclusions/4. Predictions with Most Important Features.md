- Precision (0 & 1): Measures how many predicted samples for each class are correct. High precision for class 1 shows the model predicts AAFD cases accurately; low precision for class 0 in early subsets indicates many false positives for non-AAFD.
- Recall (0 & 1): Measures how many actual samples were correctly identified. High recall for class 1 shows most AAFD cases are detected; low recall for class 0 indicates many non-AAFD cases are missed initially.
- F1 (0 & 1): Harmonic mean of precision and recall. Reflects balance between detecting and predicting each class. Early subsets have poor F1 for class 0, improving slightly with additional features.
- Macro F1: Average of F1 scores across classes, treating them equally. Shows overall model balance.
- Weighted F1: Average F1 weighted by class support. Indicates overall model performance considering class distribution.
- AUC: Measures the ability to distinguish between classes. Values close to 1 indicate excellent discrimination, while low values reflect poor separation for that subset.
